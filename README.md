# BFunBot Skills

Plug-and-play skills for AI agents to interact with BFunBot — the on-chain token creation and AI platform on BSC.

## Install a skill

Tell your OpenClaw agent:

```
install the bfunbot skill from https://github.com/BFunBot/skills/tree/main/bfunbot
```

## Available Skills

| Skill | Description |
|-------|-------------|
| [bfunbot](./bfunbot) | Create tokens on BSC, check earnings, manage BFun.bot Credits, and use BFunBot as your AI provider |
